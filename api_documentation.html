Flask Search API Documentation

1. Search

Endpoint to find the most similar name based on the input sentence.

Endpoint: /search
Method: POST
Request Body:
{
    "sentence": "input sentence"
}
Response Codes:
- 200: Success
- 404: No similar name found above the threshold
- 400: Bad request (e.g., missing input sentence)
- 500: Internal server error

Response Example (Success):
{
    "success": 200,
    "data": {
        "name": "found name",
        "geometry": "geometry data",
        "layer_id": "layer ID"
    }
}

Response Example (Error):
{
    "error": 404,
    "data": "No similar name found above the threshold."
}

2. Is Service Alive

Endpoint to check if the service is running.

Endpoint: /is_service_alive
Method: GET
Response Codes:
- 200: Service is alive
- 500: Internal server error

Response Example:
{
    "success": 200,
    "data": "Service is alive"
}
