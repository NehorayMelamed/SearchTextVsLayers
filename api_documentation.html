Flask Search API Documentation

1. Search

Endpoint to process a list of sentences and find the most similar names for each sentence, along with their geometries and layer IDs.

Endpoint: /search
Method: POST
Request Body:
{
    "sentences": [
        "First sentence to search",
        "Second sentence to search",
        ...
    ]
}
Response Codes:
- 200: Success
- 400: Bad request (e.g., no input sentences provided)
- 500: Internal server error

Response Example (Success):
{
    "status": "success",
    "data": [
        {
            "response_status": {
                "code": "Success",
                "details": null
            },
            "data": {
                "name": "found name for first sentence",
                "geometry": "POINT (34.22999264890063 31.32897455472972)",
                "layer_id": "layer ID"
            }
        },
        {
            "response_status": {
                "code": "Error",
                "details": "No similar name found above the threshold"
            },
            "data": {
                "geometry": null,
                "layer_id": null,
                "name": null
            }
        },
        ...
    ]
}

Response Example (Error):
{
    "status": "error",
    "data": "Error message"
}

2. Is Service Alive

Endpoint to check if the service is running.

Endpoint: /is_service_alive
Method: GET
Response Codes:
- 200: Service is alive
- 500: Internal server error

Response Example:
{
    "status": "success",
    "data": "Service is alive"
}
